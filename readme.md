# dotfiles
a place to store and share my dotfiles

## prerequisite
need the followings:
- terminal - macos included
- zsh - macos included
- [git](https://git-scm.com/) - macos included
- [brew](https://brew.sh/)
- [idea](https://www.jetbrains.com/idea/) - as well as the [ideavim](https://github.com/JetBrains/ideavim) plugin
- [neovim](https://neovim.io/)

## quick installation
clone this repo as
```sh
~/Developer/dotfiles
```

keep clam and run the installation script
```sh
cd ~/Developer/dotfiles && ./install.sh
```
